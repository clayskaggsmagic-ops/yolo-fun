# ---------- yolo-fun: MJPEG Streaming Server ----------
# Tested on Raspberry Pi 4B / 64-bit Bookworm
#
# ⚠️  ALL dependencies must come from APT, not pip.
#     pip wheels for Python 3.13 on aarch64 are compiled with ARMv8.2
#     instructions that cause SIGILL on the Pi 4's ARMv8.0 Cortex-A72.
#     System apt packages are compiled for the exact hardware.
#
# Install everything in one command:
#
#   sudo apt install -y python3-flask python3-opencv python3-numpy python3-picamera2
#
# Then run with the SYSTEM Python:
#
#   python3 app.py
#
# Do NOT use pip install, do NOT use Python 3.13.
# Use the system python3 (Python 3.11 on Bookworm).
